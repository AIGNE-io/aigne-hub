# AI-Kit Lib

AI-Kit common UX component

## Usage

```jsx
import React from 'react';
import { HelloWorld } from '@blocklet/ai-kit';

function App() {
  return <HelloWorld />;
}
```

## Develop

```

# develop with @blocklet/ai-kit and storybook
npm run dev

# only develop with @blocklet/ai-kit
npm run watch

# only develop with storybook
npm run storybook

```
