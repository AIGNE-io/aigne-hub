# 发布说明

本文档详细记录了 AIGNE Hub 的所有重要变更、新功能和错误修复。DevOps 和 SRE 团队可利用这些信息来跟踪平台的演进，了解每次更新的影响，并据此安排升级。

## 2025 年 10 月

### 新功能

*   **视频生成支持**：平台的核心能力已扩展，现支持视频生成模型。`ModelCalls` 表中已新增 `video` 类型，用于跟踪这些请求。
*   **媒体时长分析**：为加强多媒体内容的使用情况跟踪，`Usages` 表中已添加 `mediaDuration` 字段。这使得对音频和视频生成的成本及消耗分析更为精确。
*   **请求追踪**：所有模型调用中均已添加 `traceId` 字段。该标识符通过支持跨不同系统组件对请求进行一致性跟踪，提升了系统的可观察性，从而简化了调试和性能分析。

## 2025 年 9 月

### 新功能

*   **用于负载均衡的凭证权重**：AI 提供商凭证中已添加 `weight` 属性。此功能实现了加权负载均衡，允许管理员根据预设优先级在多个密钥之间分配 API 流量，例如将更多请求路由到更高额度或更低成本的凭证。
*   **凭证错误跟踪**：`AiCredentials` 表中已添加 `error` 字段。这使系统能够存储与特定凭证相关的最后一个已知错误，从而改进了诊断过程，并使识别和解决提供商 API 密钥问题变得更加容易。

## 2025 年 8 月

### 新功能

*   **模型健康状况监控**：引入了一套新系统，用于跟踪各个 AI 模型的运行状态。`AiModelStatuses` 表现在会记录每个模型的健康状况和可用性，从而更好地洞察提供商方面的问题，并提高整体系统的可靠性。
